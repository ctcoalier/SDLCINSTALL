Claude News

Short, dated notes on what's changing with Claude and Claude Code — written by the group, for the group. Always check the official source links for the authoritative details.

June 18, 2026

Opus 4.8 leads the latest Claude lineup

The current Claude family centers on the 4.X models — with Opus 4.8 as the most capable tier alongside Sonnet and Haiku options. For Claude Code users, that means stronger long-context reasoning for big refactors. Worth re-checking which model your workflows default to.

Source: anthropic.com/news

June 10, 2026

Skills & slash commands keep expanding in Claude Code

Skills let you package repeatable workflows that Claude Code can invoke on demand, and the plugin/marketplace ecosystem keeps growing. If you haven't tried authoring a custom skill yet, it's a great topic to bring to the next meetup — see our Dev Guides for a starting point.

Source: docs.anthropic.com

May 28, 2026

MCP continues to standardize tool connections

The Model Context Protocol (MCP) gives Claude a consistent way to talk to external tools and data sources. More servers — from browsers to databases — are becoming plug-and-play. A handy way to extend what Claude Code can do without custom glue code.

Source: modelcontextprotocol.io

May 15, 2026

Claude Code is everywhere: terminal, desktop, web, and IDEs

Claude Code runs as a CLI, in the desktop app, on the web, and via IDE extensions for VS Code and JetBrains. Different surfaces fit different workflows — bring your favorite setup to share at a meetup.

Source: anthropic.com
